Compose complex, data-driven visualizations from reusable charts and components with d3.

  • Layout charts and components automatically
  • Powerful foundation for creating custom charts and components

Getting Started

  • Download
  • Add Chart
  • Add Component
  • Add Scales

1. Download

Download and add d3.compose (and its dependencies: d3, d3.chart, and underscore) to your page.

{% highlight html %} d3.compose
{% endhighlight %}

2. Add Chart

Add chart...

{% highlight js %} var data = [{x: 0, y: 0}, {x: 1, y: 10}, {x: 2, y: 40}, {x: 3, y: 90}]; var chart = d3.select('#chart').chart('Compose', function(data) { return { charts: { lines: { type: 'Lines', data: data } } }; }); chart.draw(data); {% endhighlight %}

3. Add Component

Add component...

{% highlight js %} var chart = d3.select('#chart').chart('Compose', function(data) { return { charts: { lines: {type: 'Lines', data: data} }, components: { yAxis: { type: 'Axis', position: 'left', scale: {domain: [0, 90]} } } }; }); {% endhighlight %}

4. Add Scales

Add scales...

{% highlight js %} var chart = d3.select('#chart').chart('Compose', function(data) { var scales = { x: {data: data, key: 'x'} y: {domain: [0, 100]} }; return { charts: { lines: {type: 'Lines', data: data, xScale: scales.x, yScale: scales.y} }, components: { xAxis: { type: 'Axis', position: 'bottom', scale: scales.x }, yAxis: { type: 'Axis', position: 'left', scale: scales.y } } }; }); {% endhighlight %}
